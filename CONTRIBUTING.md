# Contributing

Contributions are welcome! Here's how to get started:

## Setup

```bash
cd salah-calendar
pip install requests
# or with uv:
uv sync
```

## Regenerate the calendar

```bash
python generate.py
```

This fetches prayer times from the AlAdhan API and generates `malmo-sweden.ics`.

## Project structure

```
salah-calendar/     # Python generator
  generate.py       # Main generation script
  main.py           # Entry point
  pyproject.toml    # Package config
docs/               # GitHub Pages site
  index.html        # Landing page
  manifest.json     # PWA manifest
  malmo-sweden.ics  # Published calendar (generated by CI)
.github/workflows/  # CI/CD
  deploy.yml        # Regenerates calendar + deploys to Pages
```

## Guidelines

- Prayer times come from [api.aladhan.com](https://aladhan.com/prayer-times-api)
- The `.ics` file must be RFC 5545 compliant
- Islamic holiday dates are approximate — verify with local community
- Test that the generated `.ics` imports correctly into Apple Calendar and Google Calendar

## Reporting issues

If prayer times seem incorrect for your location, open an issue with:
- Your city and coordinates
- The expected vs. actual prayer time
- Which calculation method your local masjid uses
